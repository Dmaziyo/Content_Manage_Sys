# Content_Manage_Sys
一个前后端的内容管理系统
### 建议使用淘宝源
### node_server
- 首先修改node_server文件中config路径下的config.default.js
- 改成自己的数据库名称以及自己数据库的名称和密码
![image](https://user-images.githubusercontent.com/74136983/160832259-a3a71fac-7397-497a-9b49-83b80082747e.png)
```node
npm install
npm run dev
```
### client
```node
yarn install
yarn run serve
```
